# Playnite Extension

This folder contains the GameKeeper export extension for Playnite.

## Installation

See [`playnite-extension/README.md`](./playnite-extension/README.md) for detailed installation instructions on your Windows PC.

## Quick Install

1. Copy the `playnite-extension` folder to: `%AppData%\Playnite\Extensions\GameKeeperExport\`
2. Restart Playnite
3. Use: **Extensions** → **GameKeeper** → **Export for GameKeeper**

## What It Does

Exports only Epic, GOG, and Xbox games from Playnite to a minimal JSON file (~100-500 KB instead of 10+ MB).
